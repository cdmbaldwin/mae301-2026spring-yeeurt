# MAE301 Course Project - Team YEEURT

This repository contains the work for the MAE301 Course Project by Team YEEURT.

## Project Structure

*   **`proposal/`**: Contains the idea pitch markdown and video link.
*   **`nanogpt/`**: Houses the nanoGPT implementation, training code, and report.
*   **`mvp/`**: Contains the Minimum Viable Product code, demo instructions, and report.
*   **`docs/`**: For any additional documentation.

## Project Timeline & Milestones

### Phase 1 – Idea Pitch (Due: Mar 16)
*   1-page idea markdown and a 5-minute video.

### Phase 2 – nanoGPT Build (Due: Apr 13)
*   GPT-style language model implementation, training runs, validation, and report.

### Phase 3 – MVP Build (Due: Apr 27)
*   Development of an MVP for the AI startup idea and accompanying report.
